## Installation

Add the following to your `package.json`:

```json
{
  "devDependencies": {
    "slidev-theme-neon": "file:./slidev-theme-neon"
  }
}
```

Then run:

```bash
npm install
# or
pnpm install
```

## Usage

In your slides markdown file:

```yaml
---
theme: neon
---
```
